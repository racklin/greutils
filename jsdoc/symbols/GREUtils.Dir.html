<static> {Boolean} GREUtils.Dir.contains(aPath, aFile)
Tests whether a file is a descendant of a directory. This method will also return false if either file or the directory does not exist.
Parameters:
{String} aPath
This is the directory to test
{String} aFile
This is the full path of the file to test
Returns:
{Boolean} Returns true if the file is a descendant of the directory, false otherwise
<static> {nsILocalFile} GREUtils.Dir.create(aPath)
Creates a new file at the given path with default permissions and returns an nsILocalFile instance representing the file. If the file already exists, that file is returned. If the file cannot be created, null is returned.
Parameters:
{String} aPath
This is the file path
Returns:
{nsILocalFile} The file, or null if the file does not exist and is not/cannot be created
<static> {nsILocalFile} GREUtils.Dir.getFile(aPath, autoCreate)
Returns an nsILocalFile instance representing the given file. Bt setting autoCreate to "true", this method will attempt to create the file with default permissions if the file does not already exists. If the file does not exist and is not/cannot be created, null will be returned.
Parameters:
{String} aPath
This is the file path
{Boolean} autoCreate
This flag indicates whether to create the file if it does not exist; false by default
Returns:
{nsILocalFile} The given file, or null if the file does not exist and is not/cannot be created
<static> {Object} GREUtils.Dir.readDir(aPath, bRecursive)
Reads directory entries recursively. This method returns the directory entries as an array. Each array element can be a string representing a file, or an array representing a sub-directory. Directory entries are read recursively. If the given path is not a directory an empty array is returned.
Parameters:
{String} aPath
This is the directory path
{Boolean} bRecursive
Recursive directory default false
Returns:
{Object} Returns the directory entries as an array of strings containing file paths
<static> {Number} GREUtils.Dir.remove(aPath, aRecursive)
Removes a directory. If "aRecursive" is false, then the directory must be empty before it can be deleted.
Parameters:
{String} aPath
This is the directory path
{Boolean} aRecursive
This flag indicates if directory is to be deleted if it is not empty
Returns:
{Number} void : directory is successfully removed -1 : path does not exist -2 : aPath is not a directory -3 : delete fails
