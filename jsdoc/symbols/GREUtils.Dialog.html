<static> {Boolean} GREUtils.Dialog.prompt(dialogTitle, dialogText, input)
Displays a dialog prompting for text input. Shows a top-level prompt dialog with an edit field, OK and Cancel buttons, a window title, and prompt text. The edit field will have an initial value taken from the "value" property of the object given by the "input" parameter. If the user clicks OK, the value from the edit field is stored in the "value" property of the "input" object before returning.
Parameters:
{String} dialogTitle
This is the title of the prompt dialog
{String} dialogText
This is the prompt text
{Object} input
This object holds the value of the edit field
Returns:
{Boolean} "true" if OK is clicked, and "false" if Cancel is clicked
<static> {Boolean} GREUtils.Dialog.select(dialogTitle, dialogText, list, selected)
Displays a select dialog prompting user to make a selection. Shows a top-level select dialog with a list box of strings, OK and Cancel buttons, a window title, and prompt text. If the user clicks OK, the "value" property of the "selected" object will be set to the index of the the selected item before returning.
Parameters:
{String} dialogTitle
This is the title of the select dialog
{String} dialogText
This is the prompt text
{String[]} list
This is an array of strings for selection
{Object} selected
This object holds the index of the selected item
Returns:
{Boolean} "true" if OK is clicked, and "false" if Cancel is clicked
