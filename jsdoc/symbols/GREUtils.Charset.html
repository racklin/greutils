<static> {String} GREUtils.Charset.convertCharset(text, in_charset, out_charset)
Converts a string from one character set encoding to another. This method takes a string encoded in encoding "in_charset" and returns the corresponding string in encoding "out_charset".
Parameters:
{String} text
This is the string to convert
{String} in_charset
This is the character set encoding of the string
{String} out_charset
This is the character set encoding to convert the string to
Returns:
{String} A string encoded in "out_charset" if conversion succeeds; otherwise the original string is returned
<static> {String} GREUtils.Charset.convertFromUnicode(text, charset)
Converts a Unicode string to the given character set encoding. This method takes a string encoded in Unicode, and returns the corresponding string encoded in the given character set encoding. If a character set encoding is not specified then the string will be converted to "UTF-8".
Parameters:
{String} text
This is the Unicode string to convert
{String} charset
This is the character set encoding to convert to
Returns:
{String} The string in the given encoding if conversion succeeds; otherwise the original string is returned

Converts a string to Unicode. This method takes a string encoded in the character "charset", and returns the corresponding string encoded in Unicode. If "charset" is not given then string is assumed to be encoded in "UTF-8".
Parameters:

{String} charset
