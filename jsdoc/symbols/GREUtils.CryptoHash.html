Namespace GREUtils.CryptoHash

GREUtils.CryptoHash This is a set of Javascript wrappers around the XPCOM nsiCryptoHash component that provides native implementation of cryptographic hash function. This can be used, for example, to calculate the MD5 hash of a file to determine if it contains the data you think it does. The hash algorithms supported are MD2, MD5, SHA-1, SHA-256, SHA-384, and SHA-512.
Defined in: GREUtils.js.
